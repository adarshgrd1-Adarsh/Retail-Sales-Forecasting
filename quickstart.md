## Quickstart – Interactive Sales Forecasting Dashboard

This folder contains a Streamlit-based **Retail Sales Forecasting Dashboard** plus helper scripts and docs that match your project proposal.

---

### 1. Files

- `app.py` – Main Streamlit dashboard (with animations and KPIs).
- `requirements.txt` – Python dependencies.
- `prepare_data.py` – Script to convert UCI Online Retail Excel to daily sales CSV.
- `data/online_retail_daily.csv` – (optional, generated by `prepare_data.py`).

---

### 2. Install dependencies

In PowerShell:

```powershell
cd "C:\Users\hp\New folder"
python -m pip install --upgrade pip
pip install -r requirements.txt
```

If `streamlit` isn’t on PATH, we will always run it via `python -m streamlit`.

---

### 3. (Optional) Use real UCI Online Retail data

1. Download `Online Retail.xlsx` from the UCI repository.
2. Save it here:

   `C:\Users\hp\New folder\data\Online_Retail.xlsx`

3. Run the preparation script:

```powershell
cd "C:\Users\hp\New folder"
python prepare_data.py
```

This creates `data/online_retail_daily.csv` with columns `date`, `sales`, `category`.

If this file is missing, the app will automatically use **synthetic** but realistic data instead.

---

### 4. Run the dashboard

From the same folder:

```powershell
cd "C:\Users\hp\New folder"
python -m streamlit run app.py
```

Then open the URL shown in the terminal (usually `http://localhost:8501`).

Use the sidebar to change **category, date range, forecast horizon, and model**, and the main tabs to see **animated forecasts**, **weekly/monthly trends**, and **modeling notes** for your report.
